# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [2.0.1] - 2026-05-05

### Changed

- Bump `@eslint-react/eslint-plugin` from 4.2.3 to 5.7.2.
- Bump `typescript-eslint` to 8.59.1.

## [2.0.0] - 2026-04-15

### Changed

- **BREAKING:** Require ESLint 10 (bump from 9.x to 10.2.0).
- **BREAKING:** Migrate from `eslint-plugin-react` to `@eslint-react/eslint-plugin`.
- Bump `eslint-plugin-simple-import-sort` from 12.x to 13.0.0.
- Bump `typescript` from 5.9 to 6.0.

### Added

- CI, Dependabot, and automerge workflows.
- Formatting checks and `.editorconfig`.

## [1.0.0] - 2026-02-27

### Added

- Initial release of `@invinite/eslint-config`.
- Unified flat-config ESLint setup for Node and React TypeScript projects.
- Bundled Prettier compatibility, import sorting, and unused-import detection.
- `node`, `react`, and base config entry points.
